The Capability APIs are a powerful set of APIs that let you do deep integration with Sense with a web developer skillset. These APIs allow you to leverage standard web technologies to embed elements of your Qlik Sense app into a page, or even into your web app logic.

A few examples of this are shown below. The full script for this page can be found in "script.js" within the package.

1 - Embed an object (which exists in a Qlik Sense app)

We can reference an object which exists in the loaded Qlik Sense app using it's GUID via the Capability API (App) with the code below:
app.getObject("qsChart1", "fzrxtjq");
> This replaces the "qsChart1" HTML element in the template with the Qlik Sense object "fzrxtjq".

2 - Embed one object (which exists), then embed a second object (which doesn't yet exist)

As above, to refence an object which already exists in the app:
app.getObject("qsChart2", "NaKQwM");
